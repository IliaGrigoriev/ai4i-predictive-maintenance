# AI4I Predictive Maintenance

A machine learning case study based on the AI4I 2020 Predictive Maintenance dataset. The project explores machine failure prediction, feature importance analysis, and model interpretability using Random Forests.

## Overview

The goal of this project is to identify the factors that contribute most to machine failures and evaluate the predictive performance of a classification model.

The workflow includes:

- Exploratory Data Analysis (EDA)
- Data preprocessing
- Random Forest model training
- Feature importance analysis
- Permutation importance analysis
- Model evaluation and reporting

## Dataset

This project uses the AI4I 2020 Predictive Maintenance Dataset, which contains operational and process-related measurements from industrial equipment.

Typical features include:

- Air Temperature
- Process Temperature
- Rotational Speed
- Torque
- Tool Wear

Target:

- Machine Failure (binary classification)
